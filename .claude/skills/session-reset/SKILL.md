# 세션 정리 모드

> AI의 금붕어 기억 문제를 방지한다.

## 설계 철학

- AI는 대화가 길어지면 초기 맥락을 잃는다
- 가정이 누적되면 잘못된 방향으로 확신한다
- 주기적으로 "리셋"하여 현재 상태를 명확히 한다

---

## 사용 시점

- 대화가 **20턴 이상** 지속되었을 때
- 작업 방향이 **여러 번 수정**되었을 때
- AI의 응답이 **이전 맥락과 모순**될 때
- 새로운 큰 작업을 시작하기 전

---

## 실행 절차

### 1. 현재 목표 요약

```
🎯 현재 목표
{지금 달성하려는 것 1~2줄}
```

불명확하면 사용자에게 확인한다.

### 2. 결정된 사항 정리

이 대화에서 합의된 내용을 나열한다:

```
✅ 결정 사항
1. {결정 1}
2. {결정 2}
3. {결정 3}
```

### 3. 완료된 작업 정리

```
📦 완료
- {작업 1} — {결과/커밋}
- {작업 2} — {결과/커밋}
```

### 4. 남은 작업 정리

```
📋 남은 작업
1. {작업} — {상태: 진행중/미시작}
2. {작업} — {상태}
```

### 5. 폐기할 가정

대화 중 만들어졌지만 더 이상 유효하지 않은 가정을 제거한다:

```
🗑 폐기
- {가정 1} — 이유: {왜 더 이상 유효하지 않은가}
```

없으면 생략.

### 6. 정리 완료

```
🔄 세션 정리 완료
- 목표: {한 줄}
- 결정: {N}건
- 완료: {N}건
- 남은 작업: {N}건
- 다음 액션: {바로 해야 할 것}
```
